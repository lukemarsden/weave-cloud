The app: boot microservices locally, check app in visual management dashboard

In this first scenario, we'll take a look at my application, which is a socks shop, by booting it up in Docker on my laptop, using Docker for Mac and Docker Compose.

Goal: run the app on my laptop, and check that it’s working

You can find the Docker Compose file here. Start it up like this:

$ git clone \
  https://github.com/microservices-demo/microservices-demo
$ cd deploy/docker-only
$ docker-compose up -d

Then plug it into a Weave Cloud account and run: 

$ sudo curl -L git.io/scope -o /usr/local/bin/scope
$ sudo chmod a+x /usr/local/bin/scope
$ scope launch --service-token=YOUR_TOKEN
